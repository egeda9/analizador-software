# Ejemplos de Especificaciones de Software

## Ejemplo 1: Sistema de Gestión de Biblioteca

Necesito un sistema de gestión de biblioteca que permita:
- Registrar libros con ISBN, título, autor, editorial y fecha de publicación
- Gestionar préstamos de libros a usuarios registrados
- Controlar las fechas de devolución y renovaciones
- Generar multas automáticas por retrasos en la devolución
- Buscar libros por título, autor, ISBN o categoría
- Reservar libros que estén actualmente prestados
- Generar reportes de libros más prestados y usuarios más activos

## Ejemplo 2: Sistema de Punto de Venta (POS)

Requiero un sistema de punto de venta para una tienda retail que incluya:
- Registro de productos con código de barras, precio, stock
- Procesamiento de ventas con múltiples formas de pago
- Control de inventario en tiempo real
- Gestión de clientes y programa de puntos
- Emisión de facturas electrónicas
- Devoluciones y cambios de productos
- Reportes de ventas diarias, semanales y mensuales
- Alertas de bajo stock
- Gestión de usuarios y permisos

## Ejemplo 3: Sistema de Reservas de Hotel

Necesito un sistema de gestión hotelera que permita:
- Gestionar habitaciones con tipos, precios y disponibilidad
- Realizar reservas online y presenciales
- Check-in y check-out de huéspedes
- Facturación y procesamiento de pagos
- Gestión de servicios adicionales (lavandería, restaurante, spa)
- Control de limpieza y mantenimiento de habitaciones
- Reportes de ocupación y revenue
- Integración con sistemas de pago en línea
- Notificaciones automáticas por email/SMS

## Ejemplo 4: Sistema de Gestión de Proyectos

Requiero una plataforma de gestión de proyectos ágil que incluya:
- Creación y seguimiento de proyectos y sprints
- Gestión de tareas con estados (to-do, in progress, done)
- Asignación de tareas a miembros del equipo
- Tableros Kanban interactivos
- Sistema de comentarios y adjuntos en tareas
- Seguimiento de tiempo invertido en cada tarea
- Generación de reportes de productividad
- Calendario de eventos y deadlines
- Notificaciones en tiempo real
- Integración con herramientas de desarrollo (Git, CI/CD)

## Ejemplo 5: Sistema de Gestión de Clínica Médica

Necesito un sistema de gestión para una clínica médica que permita:
- Registro de pacientes con historial médico completo
- Agendamiento de citas con disponibilidad de médicos
- Gestión de consultas y diagnósticos
- Prescripción de medicamentos y tratamientos
- Control de exámenes de laboratorio y resultados
- Facturación de servicios médicos
- Gestión de seguros médicos
- Reportes estadísticos de atenciones
- Control de inventario de medicamentos e insumos
- Sistema de recordatorios de citas

## Ejemplo 6: Sistema de E-Learning

Requiero una plataforma de educación en línea que incluya:
- Creación y gestión de cursos multimedia
- Sistema de inscripción y pagos
- Reproducción de videos con control de progreso
- Evaluaciones y quizzes con calificación automática
- Foros de discusión por curso
- Mensajería entre estudiantes y profesores
- Certificados de finalización automáticos
- Dashboard de progreso del estudiante
- Sistema de recomendaciones de cursos
- Reportes de desempeño académico

## Cómo Usar Estos Ejemplos

1. Copia cualquiera de estas especificaciones
2. Pégala en el cuadro de texto de la aplicación
3. Haz clic en "Analizar con IA"
4. La IA generará automáticamente los procesos, subprocesos y casos de uso

También puedes modificar estos ejemplos o crear tus propias especificaciones personalizadas.
